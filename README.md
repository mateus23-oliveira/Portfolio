## Como rodar localmente:
1. Clone o repositório:
```bash
  git@github.com:mateus23-oliveira/Portfolio.git
  cd seu-repositorio
```
2.Instale as dependências:
```bash
npm install
```
3.Execute o projeto:
```bash
npm run dev
```
